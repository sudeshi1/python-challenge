## Accomplishments
### Task 1 - PyBank
Created a Python script for analyzing financial records for a company.

![PyBank](/images/bank1.gif)

**Tasks:**
1. The total number of months included in the dataset
2. The net total amount of "Profit/Losses" over the entire period
3. Calculate the changes in "Profit/Losses" over the entire period, then find the average of those changes
4. The greatest increase in profits (date and amount) over the entire period
5. The greatest decrease in profits (date and amount) over the entire period

Lastly, printed the results to the terminal as well as exported a text file with results.

### Task 2 - PyPoll
Created a python script to analyze and modernize the vote counting process for a small, rural town

![PyPoll](/images/poll1.gif)

**Tasks:**
1. The total number of votes case
2. A complete list of candidates who received votes
3. The percentage of votes each candidate won
4. The total number of votes each candidate won
5. The winner of the election based on popular vote.

Lastly, printed the results to the terminal as well as exported a text file with results.
